型の指定は礼儀

_マイクラの用いるシェーディング言語
glsl,hlslはC言語をベースにした言語です
float,intなどのお馴染みの型に加えて
vec,matなどの色、座標(ベクトル)の計算がしやすいような型が用意されています

glslhlsl説明
float float 浮動小数点数
例)float a = 1.0;
cf. float a = 1; これはダメ
int int 整数。
例)int a = 1;
cf. int a = 1.0; これはダメ
bool bool ブーレアンの略 値はtrueまたはfalseのみ
例)bool a = true;
vec2 float2 floatが二つ集まったもの
例)vec2 a = vec2(1.0,1.0);
float2 a = float2(1.0,1.0);
cf. vec2 a = vec2(1); これはok(int→vec2への型変換)
cf. float2 a = float2(1.0); これはダメ
cf. float2 a = 1.0; hlslはfloatで入れる
vec3 float3 floatが三つ集まったもの
例)vec3 a = vec3(1.0,1.0,1.0);
float3 a = float3(1.0,1.0,1.0);
cf. vec3 a = vec3(1); これはok(int→vec3への型変換)
cf. float3 a = float3(1.0); これはダメ
cf. float3 a = 1.0; hlslはfloatで入れる
